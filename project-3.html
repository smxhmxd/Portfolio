Project Overview

In an increasingly digital world, ensuring the security of user accounts is of utmost importance. Traditional alphanumeric passwords can be vulnerable to various forms of attacks and may not provide a seamless user experience. This project has a Graphical-Based User Authentication System that uses a combination of five images as an additional password with the alphanumeric password to enhance security while offering a user-friendly authentication process.

The Graphical-Based User Authentication System aims to revolutionize the way users secure their accounts by providing a visually intuitive and highly secure authentication method. By combining the strengths of graphical elements and multi-factor authentication, this system offers robust protection against common security threats while ensuring user convenience and satisfaction.
